## 開発環境構築

npm run devで開発用サーバーを起動します。

### 動作確認

 [http://localhost:3000](http://localhost:3000) にアクセスできるか確認してください。
アクセスできたら成功です。

### 開発サーバーの停止

Ctrl + C　で開発サーバーを停止できます。

### コマンド一覧

| Make                | 実行する処理                                                            |
| ------------------- | ----------------------------------------------------------------------- |
| npm run dev        | 開発サーバーの起動 |
| Ctrl + C             | 開発サーバーを停止                                                          |
